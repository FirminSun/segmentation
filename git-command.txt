1.首先添加ssh-key，完成后在~/.ssh目录下有 id_rsa.pub 或 id_dsa.pub 文件。
a.创建ssh-key
	ssh-keygen -t rsa -C "your_email@example.com"
b.将id_rsa.pub内容复制
	clip < ~/.ssh/id_rsa.pub
c.登录网页版github->Settings->ssh and GPG keys->new ssh key
粘贴内容
d.检查是否成功：“Hi choubin! You've successfully authenticated, but GitHub does not provide shell access.”说明成功

ssh -T git@github.com

新建目录，进入目录
2.初始化：
git init
3.关联仓库：
git remote add origin git@github.com:FirminSun/segmentation.git
4.pull文件
git pull origin master --allow-unrelated-histories
5.添加文件：
git add youfile
6.提交文件
git commit -m "说明"
7.push代码到master分支：
git push -u origin master
8.删除已经add的文件（重新add）
git rm --cached "文件路径"
